# 🏭 Machine Telemetry Analytics – Factory Performance Dashboard

## 🔍 Project Overview

This project analyzes machine telemetry data from multiple factory locations to identify downtime patterns, device failures, and operational inefficiencies.

The goal is to support maintenance prioritization and improve factory performance through data-driven insights.

---

## 🛠️ Tools & Technologies

* Tableau (Dashboard & Visualization)
* JSON Data Processing
* Microsoft Excel (Data Cleaning & Structuring)

---

## 📊 Data Description

* Machine telemetry data from 4 factory locations
* Includes device type, downtime events, and failure frequency

---

## 📌 Key Analysis Performed

* Downtime frequency analysis by factory
* Device failure rate comparison
* Cross-location performance comparison
* Identification of high-risk machines

---

## 📈 Dashboard Features

* Factory-level filters
* Device-type filters
* Interactive comparison across locations
* Visual insights for maintenance decisions

## 🚀 How to Use

1. Open `.twbx` file in Tableau
2. Use filters to analyze:

   * Factory performance
   * Device failure rates
3. Explore dashboards for insights

---

## 🎯 Key Insights

* Identified factories with highest downtime frequency
* Highlighted device types with maximum failures
* Enabled comparison across multiple factory locations
